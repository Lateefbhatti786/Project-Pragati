# 🚀 Project Pragati

A web-based platform designed to empower rural women entrepreneurs through structured mentorship, multilingual accessibility, and role-based system management.

---

## 📌 Overview

Project Pragati is developed to bridge the gap between rural communities and digital support systems. It provides a centralized platform where users can access resources, mentorship, and guidance in an accessible and user-friendly manner.

---

## ✨ Features

- 🔐 User Authentication (Login/Logout with Session Management)
- 👤 Role-Based Access (Admin & User Dashboard)
- 🌐 Multilingual Support (Dynamic Language Switching)
- 📊 Admin Panel for System Control
- 💻 Responsive and Clean UI Design

---

## 🏗️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL *(if used)*  
- **Tools:** VS Code, XAMPP  

---

## 🔄 System Workflow

1. User logs in with credentials  
2. Backend validates user using PHP  
3. Session is created for authenticated access  
4. User/Admin is redirected based on role  
5. Dashboard provides access to features  
6. Language can be switched dynamically
